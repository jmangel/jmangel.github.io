Digital Linguistics

My most recent project is with some linguistics faculty and graduates at my school, UCSB, writing browser-based software for linguistics researchers to aid in tracking research data. In addition to contributing to the big-picture design discussions, one of my biggest coding tasks was to create a page of the website to demonstrate the basic function of our data structure, using mostly Javascript and the Intro.js library. Click here to see the demonstration.

I built this demo using an open-source Javascript library called Intro.js, which allows developers to show their website's features using a step-by-step tutorial. This demonstration will also show you the basic idea of the data structure that my team built. We've since been working on building a database with an API and an SDK for Javascript developers who want to create their own apps using our data structure.
